## 已修复的关键问题

### 1. FTS5 CJK 搜索 ✅
**问题**: unicode61 分词器不为 CJK 字符创建 token，导致中文搜索失败  
**解决**: 使用 trigram 分词器，自动迁移现有数据库  
**验证**: `search_finds_cjk_substring` 测试通过

### 2. 短子串搜索 ✅
**问题**: trigram 分词器要求 3 字符查询，无法搜索 1-2 字符的短子串（如 "ab"）  
**解决**: 实现混合搜索策略：
  - 短查询（< 3 字符）使用 SQL `LIKE '%query%'`
  - 长查询（≥ 3 字符）使用 FTS5 trigram
**验证**: `search_finds_short_substring` 测试通过
**示例**: 
  - 搜索 "ab" 能找到 "123ab332sddsdf" ✅
  - 搜索 "云之家" 能找到 "同步组织排序码到云之家" ✅

### 3. HTML 实体解码 ✅
**问题**: 多个裸 `&` 之间的内容丢失  
**解决**: 非 `;` 结尾的 `&` 串原样 flush  
**验证**: `html_to_plain_text_keeps_text_between_bare_ampersands` 测试通过

### 4. Rust 编译环境 ✅
**问题**: 缺少 MSVC 链接器  
**解决**: 安装 Visual Studio Build Tools  
**验证**: 所有 Rust 测试通过
